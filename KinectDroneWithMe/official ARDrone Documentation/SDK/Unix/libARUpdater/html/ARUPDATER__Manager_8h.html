get the version of a given plf file

Parameters
manager: pointer on the manager
[in]rootFolder: root folder of the plf
[in]product: the plf of the product to be tested
[in]version: the version of the remote plf
[in]edition: the edition of the remote plf
[in]extension: the extension of the remote plf
[out]localVersionBuffer: out buffer that will contain the version of the local plf
[in]bufferSize: The size of the localVersionBuffer
[out]error: ARUPDATER_OK if operation went well, the description of the error otherwise. Can be null
Returns
1 if the product plf is at the same version (or more recent) as the local plf
