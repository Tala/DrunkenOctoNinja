Fill a mission item with the given params and the default params.

Parameters
[out]missionItem: pointer on the mission item to fill
[in]param1: Radius in which the MISSION is accepted as reached, in meters
[in]param2: Time that the MAV should stay inside the PARAM1 radius before advancing, in milliseconds
[in]param3: For LOITER command MISSIONs: Orbit to circle around the MISSION, in meters. If positive the orbit direction should be clockwise, if negative the orbit direction should be counter-clockwise. [in] : For NAV and LOITER command MISSIONs: Yaw orientation in degrees, [0..360] 0 = NORTH
[in]latitude: the latitude of the mission item
[in]latitude: the latitude of the mission item
[in]longitude: the longitude of the mission item
[in]altitude: the altitude of the mission item
[in]command: the command of the mission item
[in]seq: the seq of the mission item
[in]frame: The coordinate system of the MISSION. see MAV_FRAME in mavlink_types.h
[in]current: false:0, true:1
[in]autocontinue: autocontinue to next wp
Returns
ARMAVLINK_OK if operation went well, the enum description of the error otherwise
