int ARNETWORK_IOBufferParam_t::ackTimeoutMs

int32_t ARNETWORK_IOBufferParam_t::dataCopyMaxSize

Maximum Size, in byte, of the data copied in the buffer (the IOBuffer allocates a buffer of numberOfCell X dataCopyMaxSize ). A value of ARNETWORK_IOBUFFERPARAM_DATACOPYMAXSIZE_USE_MAX automatically allocates the maximum size allowed by the underlying media.
