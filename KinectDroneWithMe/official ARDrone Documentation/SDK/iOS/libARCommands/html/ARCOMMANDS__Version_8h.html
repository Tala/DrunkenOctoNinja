int ARCOMMANDS_Version_CompareVersionCodes ( const char *  v1,
const char *  v2 
)

Compare two version code by String representation. If a version code is incomplete, the remaining part is considered to be all zeros. i.e. "3" is the same as "3.0.0.0" and "3.1.2" is the same as "3.1.2.0".

Parameters
v1The first version code.
v2The second version code.
Returns
a negative integer if v1 is newer than v2, 0 if v1 and v2 are the same, or a positive integer if v2 is newer than v1.
